Cara Membuka Finder Menggunakan Pintasan Keyboard Mac

Finder adalah salah satu fitur dari MacOS yang dapat membantu pengguna menemukan dan mengatur file. Fitur ini akan selalu muncul pertama kali saat perangkat Mac di hidupkan. Ini terbuka secara otomatis dan tetap terbuka meskipun saat sedang menggunakan aplikasi lain.

Menu Finder ini akan selalu muncul di bagian atas layar dan di bawah desktop. Ini menggunakan ikon dan jendela untuk menunjukkan kepada pengguna konten Mac, iCloud Drive, dan perangkat penyimpanan lainnya.

Mac memiliki sejumlah pintasan keyboard dan mungkin sudah banyak yang menggunakan beberapa shortcut pada keyboard. namun, masih banyak juga pengguna Mac yang masih belum mengetahui shortcut ini atau tahu, tapi tidak menggunakannya.

Jika kamu akan sering mencari file atau media lainnya di Macbook, berikut adalah pintasan cepat yang memungkinkan membuka Finder menggunakan pintasan keyboard. Begini caranya.

Membuka Finder menggunakan Pintasan Keyboard di Macbook

Untuk langsung mengakses jendela pencarian Finder menggunakan pintasan keyboard, yang perlu dilakukan hanyalah menekan Cmd + alt + Spasi secara bersamaan dan jendela Finder akan terbuka dengan pencarian yang diaktifkan di dalamnya.

Untuk pengguna yang penasaran dengan cara langsung membuka Finder menggunakan pintasan keyboard Macbook maka dapat melihat tutorialnya di bawah ini.

  1. Pintasan Keyboard: Command + Alt + Spasi.

Jika tidak menyukai pintasan ini atau ingin menonaktifkannya, dapat mengunjungi pengaturan dengan masuk ke logo Apple , klik di atasnya dan menuju System Preferences dan pergi ke Keyboard.

Sekarang di sisi kiri, klik Spotlight dan hilangkan tanda centang di sebelah kanan yang bertuliskan Show Finder search window.

Dengan cara ini kamu dapat dengan mudah mencari dan membuka file atau media di Finder.

Daftar Shortcut Finder Yang Berguna

Ada berbagai shortcut lain yang bisa kamu gunakan untuk mempermudah mengakses berbagai fungsi.

  1. Command-D: Gandakan file yang dipilih.
  2. Command-E: Mengeluarkan disk atau volume yang dipilih.
  3. Command-F: Mulai pencarian Spotlight di jendela Finder.
  4. Command-I: Menampilkan jendela Dapatkan Info untuk file yang dipilih.
  5. Shift-Command-C: Buka jendela Komputer.
  6. Shift-Command-D: Buka folder desktop.
  7. Shift-Command-F: Buka jendela Terbaru, menampilkan semua file yang Anda lihat atau ubah baru-baru ini.
  8. Shift-Command-G: Buka jendela Buka Folder.
  9. Shift-Command-H: Buka folder Beranda akun pengguna macOS saat ini.
  10. Shift-Command-I: Buka iCloud Drive.
  11. Shift-Command-K: Buka jendela Jaringan.
  12. Option-Command-L: Buka folder Unduhan.
  13. Shift-Command-N: Buat folder baru.
  14. Shift-Command-O: Buka folder Dokumen.
  15. Shift-Command-P: Menampilkan atau menyembunyikan panel Pratinjau di jendela Finder.
  16. Shift-Command-R: Buka jendela AirDrop.
  17. Shift-Command-T: Menampilkan atau menyembunyikan bilah tab di jendela Finder.
  18. Control-Shift-Command-T: Tambahkan item Finder yang dipilih ke Dock (OS X Mavericks atau yang lebih baru)
  19. Shift-Command-U: Buka folder Utilitas.
  20. Option-Command-D: Menampilkan atau menyembunyikan Dock.
  21. Control-Command-T: Tambahkan item yang dipilih ke bilah sisi (OS X Mavericks atau yang lebih baru).
  22. Option-Command-P: Menyembunyikan atau menampilkan bilah jalur di jendela Finder.
  23. Option-Command-S: Menyembunyikan atau menampilkan Sidebar di jendela Finder.
  24. Command–Slash (/): Menyembunyikan atau menampilkan bilah status di jendela Finder.
  25. Command-J: Tampilkan Opsi Tampilan.
  26. Command-K: Buka jendela Hubungkan ke Server.
  27. Control-Command-A: Membuat alias dari item yang dipilih.
  28. Command-N: Buka jendela Finder baru.
  29. Option-Command-N: Buat Folder Cerdas baru.
  30. Command-T: Menampilkan atau menyembunyikan bilah tab saat satu tab terbuka di jendela Finder saat ini.
  31. Option-Command-T: Menampilkan atau menyembunyikan bilah alat saat satu tab terbuka di jendela Finder saat ini.
  32. Option-Command-V: Memindahkan file di Clipboard dari lokasi aslinya ke lokasi saat ini.
  33. Command-Y: Gunakan Tampilan Cepat untuk melihat pratinjau file yang dipilih.
  34. Option-Command-Y: Lihat slideshow Tampilan Cepat dari file yang dipilih.
  35. Command-1: Melihat item di jendela Finder sebagai ikon.
  36. Command-2: Melihat item di jendela Finder sebagai daftar.
  37. Command-3: Melihat item di jendela Finder dalam kolom.
  38. Command-4: Melihat item di jendela Finder di galeri.
  39. Command–Arrow Up: Buka folder yang berisi folder saat ini.
  40. Command–Control–Arrow Atas: Buka folder yang berisi folder saat ini di jendela baru.
  41. Command–Arrow Down: Buka item yang dipilih.
  42. Arrow Right: Buka folder yang dipilih.
  43. Arrow Left: Tutup folder yang dipilih.
  44. Command-Delete: Pindahkan item yang dipilih ke Sampah.
  45. Shift-Command-Delete: Mengosongkan Sampah.
  46. Option-Shift-Command-Delete: Kosongkan Sampah tanpa dialog konfirmasi.
